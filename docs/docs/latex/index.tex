The examples are frequently being tested on {\bfseries{Windows 11}}, {\bfseries{Nix\+OS}} (with Nix flake), {\bfseries{Linux Mint}} (virtual machine and pc) with different compilers ({\bfseries{MSVC}}, {\bfseries{gcc}}), hardware configurations (RX580 4GB GDDR5, Radeon V Carrizo 500MB) and windowing systems ({\bfseries{Win32}}, {\bfseries{X11}}, {\bfseries{Wayland}}).\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Clone and Build}{Clone and Build}}\label{index_autotoc_md3}
Open the terminal and run the following commands\+:

 

Tested on Win32 and X11.


\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ -\/-\/recursive\ https://github.com/mrsinho/shvulkan.git}
\DoxyCodeLine{cd\ shvulkan}
\DoxyCodeLine{mkdir\ build}
\DoxyCodeLine{cd\ build}
\DoxyCodeLine{cmake\ -\/DSH\_VULKAN\_BUILD\_EXAMPLES=ON\ ..}
\DoxyCodeLine{cmake\ -\/-\/build\ .}

\end{DoxyCode}




Tested on X11 and Wayland graphics servers. The Nix flake lets the system choose the default windowing system.

\begin{DoxyWarning}{Warning}
While testing with Wayland revealed no issues, there are some resizing issues when using X11.
\end{DoxyWarning}

\begin{DoxyCode}{0}

\DoxyCodeLine{}
\DoxyCodeLine{set(SH\_VULKAN\_ROOT\_DIR\ \ \ \ \ path/to/shvulkan/root/directory)}
\DoxyCodeLine{set(SH\_VULKAN\_BINARIES\_DIR\ path/your/binaries/directory)}
\DoxyCodeLine{}
\DoxyCodeLine{include(\$\{SH\_VULKAN\_ROOT\_DIR\}/shvulkan/shvulkan.cmake)}

\DoxyCodeLine{}
\DoxyCodeLine{endif()}
\DoxyCodeLine{}

\hypertarget{index_autotoc_md8}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{index_autotoc_md8}
{\ttfamily shvulkan} ships with one \href{https://github.com/mrsinho/shvulkan/tree/main/examples/src/compute/power-numbers.c}{\texttt{ {\ttfamily compute example}}}, one graphics \href{https://github.com/mrsinho/shvulkan/tree/main/examples/src/graphics/clear-color.c}{\texttt{ {\ttfamily clear color example}}}, one graphics \href{https://github.com/mrsinho/shvulkan/tree/main/examples/src/graphics/scene.c}{\texttt{ {\ttfamily scene example}}} and a \href{https://github.com/mrsinho/shvulkan/tree/main/examples/src/graphics/clear-color.c}{\texttt{ {\ttfamily headless scene}}} graphics example, which instead of presenting images to the screen it streams the graphics output with an http server.
